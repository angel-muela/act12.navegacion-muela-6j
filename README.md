# Navegacion entre paginas con flutter
# Muela Dueñez Angel Adrian 6J

lenguaje Dart Flutter, nivel princpiante, navegación entre 3 paginas utilizando rutas nombradas, desde main llamar a la pagina1, en la pagina en appbar mostrar el texto "inicio muela 6j" en color blanco de fondo naranja, iconos blancos, en body un contenedor redondeado blanco de 200 x 200 con texto negro y centrado y un boton de color azul texto blanco para seleccionar pagina 2,en la pagina 2 un appbar con texto "segunda pagina 6 j" rojo y fondo negro y los iconos en blanco, en body una imagen desde la red asi bien chida y un boton para avanzar a la pagina 3, en la pagina 3 en appbar un texto co color negro "pagina 3 el muela", color de fondo verde claro, todo en un solo archivo.
